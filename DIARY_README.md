# Diary Blog 功能说明

我已经为你的diary界面设计了一个完整的blog形式，包含现代化的布局、美观的样式和丰富的功能。

## 🎨 新功能特性

### 1. 专门的Diary布局
- **diary.html**: 单篇日记的显示布局
- **diary-archive.html**: 日记列表的显示布局

### 2. 博客风格的样式
- 现代化的卡片设计
- 响应式布局，支持移动端
- 优雅的悬停效果和过渡动画
- 清晰的层次结构和排版

### 3. 丰富的元信息显示
- 发布日期和阅读时间
- 分类和标签系统
- 相关日记推荐

## 📝 使用方法

### 创建新日记
在 `_diary/` 目录下创建新的 `.md` 文件，使用以下格式：

```markdown
---
layout: diary
title: "你的日记标题"
date: 2025-01-XX 00:00:00 +0000
excerpt: "日记摘要，会显示在列表中"
categories: diary
tags: [标签1, 标签2, 标签3]
---

你的日记内容...
```

### 支持的Front Matter字段
- `layout`: 必须设置为 `diary`
- `title`: 日记标题
- `date`: 发布日期
- `excerpt`: 摘要（可选）
- `categories`: 分类（可选）
- `tags`: 标签数组（可选）

## 🔧 自定义样式

样式文件位于 `_sass/_diary.scss`，你可以根据需要修改：

- 颜色主题
- 字体大小和行高
- 间距和边距
- 动画效果

## 📱 响应式设计

- 桌面端：完整的布局和功能
- 平板端：适中的布局调整
- 移动端：优化的移动体验

## 🚀 快速开始

1. 确保你的Jekyll服务器正在运行
2. 访问 `/diary/` 页面查看日记列表
3. 点击任意日记标题查看详细内容
4. 在 `_diary/` 目录下添加新的日记文件
